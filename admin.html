♻ Translation bench

Translate any English through the live pipeline (Google Translate → glossary check → Georgian validation) and judge the result. Nothing is cached automatically — promote good rows. One string per line.

☷ Georgian glossary

Forced renderings pin a term's exact Georgian and short-circuit the translator entirely when a whole title matches (instant, never flagged). Keep-as-is terms are protected when a whole string equals one — Google Translate already leaves most proper nouns alone, so add a forced rendering for anything it gets wrong. Use Scan cache to find entries that don't follow the glossary.
